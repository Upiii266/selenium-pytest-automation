# Selenium Pytest Automation

Project automation testing menggunakan Selenium + Pytest.

## 📦 Tech Stack
- Python
- Selenium
- Pytest

## 📁 Project Structure
- pages/        → Page Object Model
- tests/        → Test cases
- screenshots/  → Screenshot hasil test (negative case)
- conftest.py   → Pytest config & fixtures
- pytest.ini    → Pytest configuration

## ▶️ How to Run
```bash
pip install -r requirements.txt
pytest
